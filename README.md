# NeoJaundice360

Comprehensive educational app for learning about newborn jaundice.

## Features

- **Evidence-Based Content**: Aligned with AAP 2022+ guidelines
- **Interactive Tools**:
  - Bhutani Nomogram Explorer
  - Phototherapy Calculator
  - Case-Based Learning
- **Comprehensive Modules**: From physiology to management
- **Mobile-First Design**: Fully responsive and optimized for mobile devices
- **Dark Mode**: Optimized for low-light environments
- **Offline Capable**: PWA with service worker support

## Technology Stack

- React 18 with TypeScript
- Vite for build tooling
- Tailwind CSS for styling
- Recharts for data visualization
- React Router for navigation
- Zustand for state management
- PWA support for offline functionality

## Getting Started

### Prerequisites

- Node.js (v18 or higher)
- npm or yarn

### Installation

```bash
# Install dependencies
npm install

# Run development server
npm run dev

# Build for production
npm run build

# Preview production build
npm run preview
```

## Educational Purpose

This application is designed for educational purposes to help healthcare trainees learn about neonatal jaundice. It is **not intended to provide medical advice** or replace clinical judgment. Always follow your institution's protocols and consult with experienced clinicians for patient care decisions.
